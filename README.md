# 🌟 Vendor Analysis Project 🌟

Welcome to my **Vendor Analysis Project** repository! 🚀  
This project focuses on analyzing vendor performance, purchasing trends, and sales insights using **Python, SQL, and Power BI**.  

It combines **data cleaning, exploratory analysis, database querying, and dashboard visualization** into one complete business intelligence project. 📊✨

---

## 📂 Repository Structure

This repository contains **4 main files/projects**:

### 📁 1. Vendor Kaggle File
- Original dataset source from Kaggle 📦  
- Raw data used for analysis and transformation  

### 🐍 2. Vendor Python Analysis
- Data cleaning with **Pandas**
- Feature engineering
- Exploratory Data Analysis (EDA)
- KPI calculations
- Trend analysis

### 🗄️ 3. Vendor SQLite / SQL Analysis
- SQL queries for vendor insights
- Revenue & sales reports
- Ranking vendors by performance
- Data extraction and filtering

### 📊 4. Vendor Dashboard (Power BI)
Interactive dashboard including:

- 💰 Total Sales  
- 📈 Growth Trends  
- 🏆 Top Vendors  
- 📦 Purchase Analysis  
- 📍 Regional Performance  
- 📅 Monthly Insights
